Overview
--------
Welcome to the Serenity SDK for Python. The Serenity digital asset platform
from `Cloudwall <https://cloudwall.tech>`_ can be used to take complex portfolios
of digital assets and break down the risk, compute Value at Risk (VaR) and 
perform other sophisticated analysis to help institutional investors gain 
insights and better understand the risks of their portfolios.

Installation
------------
You can install the latest version of the Serenity SDK from PyPi as follows:

.. code-block::

    pip install serenity.sdk.python
